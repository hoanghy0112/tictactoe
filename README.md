# Tic Tac Toe Game

## Requirements
- Git
- Cmake
- Raylib library
- Compiler C++ such as: Visual Studio, MinGW / GCC,...

## Build project
- Clone repository
```
git clone https://github.com/hoanghy0112/tictactoe.git -b ai
cd tictactoe
```
- Build project
```
mkdir build
cd build
cmake ..
cmake --build .
```
